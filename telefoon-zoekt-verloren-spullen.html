Telefoon zoekt verloren spullen

29 juli om 20:05
Lumia 625-telefoons. Â© reuters.

Nokia lanceert binnenkort een accessoire voor de Lumia-smartphones waarmee je spullen zoals sleutels kan opsporen via de telefoon.

De Treasure Tag kan via een riempje worden vastgemaakt aan een voorwerp waarvan de bezitter bang is het te verliezen. Via Bluetooth en Near Field Communication (NFC) kan een speciale app op de telefoon daarna achterhalen waar de Tag zich bevindt, meldt The Verge.

De app kan op een kaart laten zien waar de Tag is. Ook kan de telefoon piepen wanneer iemand bij zijn verloren voorwerp in de buurt komt. De Tag staat altijd 'aan' en heeft een batterijtje dat tot zes maanden meegaat.

Volgens The Verge zal de Treasure Tag binnen enkele weken op de markt komen. Dat valt samen met de release van een update voor Lumia's waardoor de toestellen Bluetooth 4.0 ondersteunen.
