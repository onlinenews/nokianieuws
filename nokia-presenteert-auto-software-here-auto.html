Nokia presenteert auto-software Here Auto

30 augustus om 10:07

Nokia heeft software voor in de auto gepresenteerd. De software draait op Nokia's voormalige cross-platform toolkit Qt. Het is de bedoeling dat autofabrikanten de software in hun nieuwe modellen gaan zetten, maar nog geen automerk heeft aangekondigd dit te gaan doen.

Here Auto is aparte software voor in de auto en geen afgeleide van andere software van Nokia voor smartphones. De software zorgt niet alleen voor navigatiefunctionaliteit, maar maakt het ook mogelijk om routes van en naar bekende plekken in te stellen om te controleren of er file staat. De dienst biedt dan zo nodig een omleiding aan. Sommige functionaliteit vereist een internetverbinding, maar veel werkt ook offline. De internetverbinding werkt via gangbare mobiele netwerken.

De software kan door fabrikanten worden ingebouwd in auto's, maar het is vooralsnog onduidelijk of dat ook zal gebeuren: Nokia zegt daarover later meer informatie te geven. Consumenten kunnen de software niet zelf downloaden. De autofabrikant kan de software middels een sdk aanpassen aan de wensen van partners.

Met de software komt een 'Here Companion'-app, een applicatie voor Windows Phone en Android waarmee routes naar de software kunnen worden gepusht en de auto in de gaten kan worden gehouden: onder meer zaken als temperatuur in de auto en hoe vol de tank is kunnen daarmee op afstand worden uitgelezen.

Een demo-model van Nokia draait op een Linux-kernel, maar ook embedded varianten van Windows kunnen bijvoorbeeld worden gebruikt.
